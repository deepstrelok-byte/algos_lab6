

# Объединение файлов по суффиксу и размеру

## Что делает

Объединяет в один файл все файлы с указанным суффиксом (например, `.txt`), размер которых **меньше** заданного значения.

## Как запустить

```bash
chmod +x script.sh
./script.sh <суффикс> <максимальный_размер> [выходной_файл]
```

## Пример

```bash
./script.sh .txt 1000 result.txt
```

Объединит все `.txt` файлы размером меньше 1000 байт в `result.txt`

## Тестовые файлы

```bash
echo "Привет мир" > file1.txt
echo "Это второй файл, он чуть длиннее предыдущего" > file2.txt
echo "Короткий файл" > file3.txt
echo "Большой файл с длинным содержимым, который не должен попасть в результат" > file4.txt
echo "Лог-файл" > test.log

# Запустить
./script.sh .txt 50 merged_result.txt

# Результат 
cat merged_result.txt
```

## Аргументы

| Аргумент | Описание |
|----------|----------|
| суффикс | Расширение файлов (например `.txt`, `.log`) |
| размер | Максимальный размер в байтах |
| выходной_файл | Имя результата (по умолчанию `merged.txt`) |
```
